# Tutorials

This folder contains step-by-step tutorials for learning how to use the Simple Autonomous Car SDK.

## Contents

- **[Costmap Tutorial](costmap_tutorial.ipynb)** - Learn how to use costmaps for obstacle representation and inflation
- **[Planner Tutorial](planner_tutorial.ipynb)** - Learn how to build and use path planning algorithms
- **[Track Bounds Alert Tutorial](track_bounds_alert.ipynb)** - Learn how to build track bounds alert systems
- **[Sensor Fusion Tutorial](sensor_fusion_tutorial.ipynb)** - Learn how to combine data from multiple sensors
- **[Performance Optimization](performance_optimization.ipynb)** - Learn how to optimize simulation performance

## Learning Path

1. Start with **Costmap Tutorial** to understand obstacle representation
2. Then **Planner Tutorial** to learn path planning
3. Learn **Sensor Fusion Tutorial** for multi-sensor systems
4. Learn **Track Bounds Alert Tutorial** for safety systems
5. Explore **Performance Optimization** for real-time applications

These tutorials provide hands-on examples with explanations of key concepts.
